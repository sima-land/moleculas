# Card
### Компонент карточки.

### Примеры использования
```jsx
import Card from '@dev-dep/moleculas/common/components/card';

// Простой вариант, с передачей только содержимого
<Card>
  <div>...</div>
</Card>
```

// Вариант с шапкой
<Card
  header={(
    <div>
        <InformationSVG />
    </div>
  )}
>
  <div>...</div>
</Card>
```
