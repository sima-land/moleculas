# Card
### Компонент карточки.

### Примеры использования
```jsx
import Card from '@dev-dep/moleculas/common/components/card';

// Простой вариант, с передачей только содержимого
<Card>
  <div>...</div>
</Card>

// Вариант с отображением шапки (заголовок + кнопка "Закрыть") и футера
<Card
  title='...'
  footer={(
    <div>...</div>
  )}
  onClose={() => { ... }}
  withCloseButton
>
  <div>...</div>
</Card>
```
