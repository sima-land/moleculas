# ListItem
### Компонент карточки товара в списке товаров.
Компонент мобильной версии. Может принимать различные варианты отображения (списком, плиткой).
Может использоваться без блока добавления в корзину. Различные варианты отображения представлены в конце документации.

### Пример использования
```jsx
import ListItem from '@dev-dep/moleculas/mobile/components/list-item';

<ListItem {...listItemProps} />
```
