# RecommendationSlider
### Компонент мобильного слайдера рекомендованных товаров.
Компонент мобильной версии. Обернут в HOC `withInViewportObserver` для загрузки элементов слайдера.

### Пример использования
```jsx
import RecommendationSlider from '@dev-dep/moleculas/mobile/components/recommendation-slider';

<RecommendationSlider
  title='Рекомендованные товары'
  items={items}
  onItemClick={() => undefined}
  onInViewport={() => undefined}
  onRequest={() => undefined}
/>
```
