# RecommendationCarousel
### Компонент карусели рекомендованных товаров.
Компонент десктопной версии. Имеет бесконечную прокрутку. Обернут в два HOC `withInViewportObserver`
для загрузки элементов карусели и отправки статистики в ОКО.

### Пример использования
```jsx
import RecommendationCarousel from '@dev-dep/moleculas/desktop/recommendation-carousel';

<RecommendationCarousel
    title='Рекомендованные товары'
    items={items}
    onItemClick={() => undefined}
    onInViewport={() => undefined}
    onRequest={() => undefined}
/>
```
