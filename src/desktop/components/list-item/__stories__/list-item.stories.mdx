# ListItem
### Компонент карточки товара в списке товаров.
Компонент десктопной версии. Может принимать различные варианты отображения (списком, плиткой).
Может использоваться без блока добавления в корзину. Различные варианты отображения представлены в конце документации.

### Пример использования
```jsx
import ListItem from '@dev-dep/moleculas/desktop/components/list-item';

<ListItem {...listItemProps} />
```
